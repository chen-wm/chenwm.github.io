# 一、新建vuepress项目

vuepress构建 (opens new window)

mkdir blog && cd blog
npm install vuepress vuepress-theme-meteorlxy
创建 `src/_posts` 文件夹和 Vuepress 配置文件，项目结构大致为：
BLOG project: 
├── src # Blog 源文件目录
│   ├── .vuepress # Vuepress 目录
│   │   └── config.js # Vuepress 配置文件
│   └── _posts # 博客文件夹
│       ├── xxx.md
│       ...
└── package.json
